# 프로젝트 기획 배경
기존 블로그와 유사하게 만들어 봄으로써 자바 기반 백엔드 역량 향상과 새로운 기술 스택을 경험해보고자 기획하였습니다. <br>
또한 무조건적으로 책과 강의를 따라하여 만드는 것이 아닌 직접 기능에 대한 구현방법을 고민하고, 여러 자료를 찾아보며 적용하는 힘을 키우기 위해 프로젝트를 시작했습니다. <br>
<br> <br>
